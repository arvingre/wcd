# WCD

WCD is the working repository for building an AI workflow orchestration and engineering collaboration system.

Current priority:

1. Verify real GitHub collaboration capability.
2. Establish the project operating structure.
3. Use GitHub as the source of truth for planning, implementation, review, and delivery.
4. Integrate local Claude Code / coding agents with repository-based workflows.

## Current Phase

**WCD-0001: Assistant Capability Validation and Project Foundation**

This phase is not about building the full product yet. It verifies which collaboration capabilities are actually available and establishes a minimal project foundation that can be used by humans and AI agents.

## Working Principle

No capability should be assumed until it is verified in the repository.

Capability states:

- Verified: tested directly in this repository.
- Supported: the tool or workflow exists, but has not been verified here.
- Not available: blocked or unavailable in the current environment.
